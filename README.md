# Placa TEB PC140C01

Placa TEB PC140C01. Não encontrei referência alguma na internet sobre esta placa, então resolvi tirar umas fotos e documentar.

## Descrição

A TEB produz eletrocardiógrafos, imagino que seja a placa de captura e processamento dos sinais.

## Componentes

- **2 DSP's** - Processadores digitais de sinal
- **4 CI's de memória RAM** - 32kb cada
- **3 PALs** - Dispositivos lógicos programáveis
- **Bus transceivers** - Controladores de barramento
- **1 Conector DB9** - Interface de comunicação
